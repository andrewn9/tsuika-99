## Run Locally

Clone the project

```bash
  git clone https:github.com/andrewn9/tsuika
```

Go to the project directory

```bash
  cd tsuika
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Tech Stack

**Client:** PIXI.js, socket.io-client

**Server:** Node, Express, socket.io
